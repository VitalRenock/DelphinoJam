Créer un nouveau processor:
- Créer un nouveau script héritant de "CommandProcessor".
- Dans le script, générer les subsitutions pour cette nouvelle classe.

Créer de nouvelles commandes pour un processor:
- Créer un nouveau script héritant de "Command".
- Dans le script, implémenter les classes abtraites pour cette nouvelle classe.
- Déclarer les nouvelles variables et/ou composants qu'on aura besoin
	pour l'action souhaitée dans le script.
- Générer le constructeur de la classe et le faire hérité de base().
- Passer dans le constructeur les valeurs requises par les variables et/ou composants pour l'action souhaitée.
